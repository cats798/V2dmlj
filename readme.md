# 自动链接弹幕v2

自动获取播放链接并从服务器获取弹幕的用户脚本。

## 功能特点

- 🎯 自动检测页面中的视频元素
- 💬 实时获取并显示弹幕
- 🎨 美观的弹幕显示界面
- 🔄 自动更新机制
- 📱 支持大多数视频网站

## 安装方法

### 前提条件
需要安装用户脚本管理器扩展：
- [Tampermonkey](https://www.tampermonkey.net/) (推荐)
- [Violentmonkey](https://violentmonkey.github.io/)
- [Greasemonkey](https://www.greasespot.net/)

### 安装步骤
1. 点击下方链接安装：
   [安装自动链接弹幕v2](https://raw.githubusercontent.com/cats798/V2dmlj/main/dmlj.js)

2. 在用户脚本管理器中确认安装

## 使用说明

1. 安装脚本后访问任何包含视频的网页
2. 脚本会自动检测视频元素并获取弹幕
3. 弹幕会显示在页面右上角的弹幕容器中
4. 点击右上角的"×"可以隐藏弹幕容器
